# Movie Recommendation System

A **content-based Movie Recommendation System** built using **Python** and **Streamlit**.  
The app recommends **5 similar movies** based on the movie selected by the user and displays their posters using the **TMDB API**.

---

## Features
- Select a movie from dropdown
- Get top 5 recommended movies
- Movie posters fetched using TMDB API
- Interactive and user-friendly Streamlit UI

---

## Technologies Used
- Python  
- Streamlit  
- Pandas  
- Requests  
- Pickle  
- Scikit-learn  
- TMDB API
